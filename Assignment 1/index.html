Would You Rather

25 Iterations of One Idea

This gallery documents the design journey of a "Would You Rather" quiz — from wild, radically different explorations to a refined final product. Each version is a fully interactive page. Click any card to try it.

Phase 1

Go Wide

The first 10 versions explored radically different visual directions — dark cards, neon arcades, magazine editorials, brutalist layouts, gradient parties, split-screen duels, newspaper tabloids, vaporwave aesthetics, terminal UIs, and bubblegum Y2K. The goal was to push the concept into surprising directions and see what resonated.

Phase 2

Converge

Versions 11–15 started mixing the strongest elements from Phase 1. The split layout met the gradients. The terminal met vaporwave. The newspaper met bubblegum. Patterns emerged: I kept gravitating toward dark backgrounds, glass effects, and clean typography. The bold type and minimal approach from v14 and the glassmorphism from v15 started defining the direction.

v11 Split + Gradient Full-screen split with purple/pink gradients, Poppins, gold accent text → Try it v12 Terminal + Vaporwave Terminal window with purple/pink glow colors, code-style layout → Try it v13 Tabloid + Bubblegum Serif headlines with pink energy, bouncing stars, gradient buttons → Try it v14 Dark Minimal Ultra-clean dark layout, Bebas Neue headers, red accent, letter keys → Try it v15 Glassmorphism Frosted glass cards, gradient text, key badges, modern app feel → Try it
Phase 3

Narrow Down

Versions 16–20 tested variations within a narrower range. Warm amber vs electric blue vs sharp editorial. By v19, the direction was clear: dark glass with gradient accents. v20 stripped everything to its essentials. The dark + glass + gold-accent direction won out.

v16 Warm Dark Centered card, amber/orange accent, warm tones on dark background → Try it v17 Neon Accent Electric blue accent, vertical stack, left-border hover, mono feel → Try it v18 Light Editorial White background, sharp square corners, black borders, red accent → Try it v19 Dark Glass Refined Glass cards with blur, purple-pink gradient, polished and app-like → Try it v20 Ultra Minimal Nearly black, single red accent, text-only options, maximum whitespace → Try it
Phase 4

Refine

The final 5 versions are all the same core design with incremental micro-refinements. v21 established the base: dark glass + gold accents + letter badges. Each version after that tweaked spacing, animations, progress indicators, and hover effects until v25 felt just right.

v21 Final Direction Dark glass, gold gradient accents, letter badges — the base → Try it v22 + Padding & Animation More card padding, larger text, fade-in transition on questions → Try it v23 + Progress Bar Swapped pips for a smooth progress bar, rounder buttons → Try it v24 + Animated Border Gradient border animation on card, scale hover, visible OR separator → Try it v25 Final Version Perfect spacing, subtle gradient background, polished results page → Try it
Final Choice

Version 25

After 24 iterations — from neon arcades to newspaper tabloids to bubblegum pink — the design converged on a dark, glassy aesthetic with gold gradient accents. It's clean, it's modern, and it lets the funny content breathe. The animated border, smooth progress bar, and polished results page make it feel like a real product.
